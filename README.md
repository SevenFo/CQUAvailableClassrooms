# CQUAvailableClassrooms
查询重庆大学教学楼的空教室


+ 从新教务网的API接口抓取全校的课程时间表数据，保存到本地数据库之后进行查询
+ 项目还在完善中，当前只有基础查询功能

#### 使用指南
```python
if __name__ == "__main__":
    tt = UniversityTimetableGeter()#实例化
    tt.init_finder()#初始化楼栋代码
    tt.get_NULL_room_by_day_db(17, 7)#查询第17周星期7，虎溪一教的空教室
```
